## Imports

```js
import { Badge } from '@tidy-ui/all';
// or
import { Badge } from '@tidy-ui/presentation';
// or
import { Badge } from '@tidy-ui/badge';
```

## Key Features

### Add Meaning to Any Component

Enhance notifications, or user statuses by wrapping any component with a **Badge**
and using the [[content]] attribute to provide additional context.

```jsx
<Stack gap="2rem">
  <Badge content={9}>
    <Icon ele={<Icon.Email />} />
  </Badge>
  <Badge content={9}>
    <Text.Body1>Mails</Text.Body1>
  </Badge>
</Stack>
```

### Custom Badge Tones

Customize the Badge's appearance with a range of [[tone]] options to match your app's color palette and branding, ensuring a cohesive design.

```jsx
<Stack gap="2rem">
  <Badge content={10} tone="major">
    <Icon ele={<Icon.Add />} />
  </Badge>
  <Badge content={10} tone="minor">
    <Icon ele={<Icon.AddCircle />} />
  </Badge>
  <Badge content={10} tone="neutral">
    <Icon ele={<Icon.CalendarMonth />} />
  </Badge>
  <Badge content={10} tone="info">
    <Icon ele={<Icon.Info />} />
  </Badge>
  <Badge content={10} tone="success">
    <Icon ele={<Icon.Success />} />
  </Badge>
  <Badge content={10} tone="warning">
    <Icon ele={<Icon.Warning />} />
  </Badge>
  <Badge content={10} tone="danger">
    <Icon ele={<Icon.Dangerous />} />
  </Badge>
</Stack>
```

### Content or Dot

Add precision by placing content or a subtle dot in the top-right corner of your component. Use the [[isDotted]] attribute to highlight key details effectively.

```jsx
<Badge isDotted tone="danger">
  <Icon ele={<Icon.Menu />} />
</Badge>
```

### Filled or Outlined Styles

Choose between a bold filled style or a more subtle outlined variant with the [[isOutlined]] attribute to match your design vision.

```jsx
<Stack gap="2rem">
  <Badge content={10} tone="major" isOutlined>
    <Icon ele={<Icon.CopyAll />} />
  </Badge>
  <Badge content={10} tone="minor" isOutlined>
    <Icon ele={<Icon.Person />} />
  </Badge>
  <Badge content={10} tone="neutral" isOutlined>
    <Icon ele={<Icon.Phone />} />
  </Badge>
  <Badge content={10} tone="info" isOutlined>
    <Icon ele={<Icon.FileUpload />} />
  </Badge>
  <Badge content={10} tone="success" isOutlined>
    <Icon ele={<Icon.CheckCircle />} />
  </Badge>
  <Badge content={10} tone="warning" isOutlined>
    <Icon ele={<Icon.Close />} />
  </Badge>
  <Badge content={10} tone="danger" isOutlined>
    <Icon ele={<Icon.Remove />} />
  </Badge>
</Stack>
```

### Numeric Limit and "+"

Display numeric content neatly with the Badge by setting a [[max]] value. When numbers exceed the limit,
the Badge adds a `+` sign for clear, concise communication.

```jsx
<Stack gap="2rem">
  <Badge content={300} max={200} tone="info">
    <Icon ele={<Icon.Email />} />
  </Badge>
  <Badge content={110} tone="info">
    <Icon ele={<Icon.Email />} />
  </Badge>
</Stack>
```

### Blinking Animation

Use the [[isBlink]] attribute to add dynamic blinking animations, creating engaging live status indicators that enhance user interaction and intrigue.

```jsx
<Badge isDotted isBlink tone="success">
  <Icon ele={<Icon.Person />} />
</Badge>
```
