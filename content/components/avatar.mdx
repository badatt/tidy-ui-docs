```js
import { Avatar, AvatarBadge, AvatarGroup } from '@tidy-ui/presentation';

import { Avatar, AvatarBadge, AvatarGroup } from '@tidy-ui/avatar';
```

## Key Features

### Avatar Display

The Avatar component provides a flexible way to represent users consistently.

#### Profile Image

Set a [[src]] to display a user's profile picture in a circular frame.

#### Initials

If no image is available, setting a [[name]] generates and displays initials.

#### Icon

When neither [[src]] nor [[name]] is provided, an [[icon]] ensures a recognizable representation.

```jsx
<Stack>
  <Avatar src="https://picsum.photos/id/64/200/200.webp" />
  <Avatar name="Isaac Newton" />
  <Avatar icon={<Icon.RotatingCircle />} />
</Stack>
```

### Default Fallback

If user details and a custom icon are unavailable, the Avatar component defaults to a universal person icon,
ensuring a consistent and visually cohesive representation.

```jsx
<Avatar />
```

### Size Customization

The Avatar component allows you to adjust its size using the [[girth]] property, offering options from extra-small
to extra-large for seamless integration with your design and UI needs.

```jsx
<Stack>
  <Avatar girth="xxs" src="https://picsum.photos/id/823/200/200.webp" />
  <Avatar girth="xs" src="https://picsum.photos/id/832/200/200.webp" />
  <Avatar girth="sm" src="https://picsum.photos/id/836/200/200.webp" />
  <Avatar girth="md" src="https://picsum.photos/id/978/200/200.webp" />
  <Avatar girth="lg" src="https://picsum.photos/id/996/200/200.webp" />
  <Avatar girth="xl" src="https://picsum.photos/id/1027/200/200.webp" />
  <Avatar girth="xxl" src="https://picsum.photos/id/1074/200/200.webp" />
</Stack>
```

### Enhanced User Representation

Elevate the Avatar component's visual impact with these features:

#### Border Accent

Add a circular border to highlight the Avatar and convey extra information.

#### Avatar Badge

Include a small dot to indicate notifications, user actions, or status updates.

#### Status Tone

Customize the badge's color to represent user status, such as online or offline.

```jsx
<Stack>
  <Avatar accent="rgb(22, 162, 73)" src="https://picsum.photos/id/646/200/200.webp" />
  <Avatar src="https://picsum.photos/id/659/200/200.webp">
    <AvatarBadge tone="rgb(245, 159, 10)"/>
  </AvatarBadge>
</Stack>
```
